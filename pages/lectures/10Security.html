Injection Attacks

Malicious users input code or commands through valid input fields

Common Types:

  • SQL Injection: Inserting SQL commands into form fields
  • Buffer Overflow: Exceeding memory boundaries
  • Command Injection: Executing system commands

Defense:

  • Input validation and sanitization
  • Parameterized queries
  • Least privilege principle
